# Python Calculator 

 This is a simple Python Calculaor project created using basic Python programming concepts.

 ## Features
 - Addition
 - subtraction
 - Multiplication
 - Division
 - Menu-driven program
 - User input based calculation

## Technologies Used
- Python 3

## How to Run the Project
1. Make sure Python is installed on your system.
2. Open or clone this repository.
3. Open terminal/Command prompt.
4. Run the file using:
Python Calculation.py

## Project Purpose
This project is created to practice Python basics such as:
- Functions
- Conditional statements
- User input/output
- Logical thinking
